# Toys Sales Analysis (SQL Project)

## Project Overview
This project analyzes toy store sales data using SQL to uncover business insights such as:

- Total Revenue
- Best Selling Products
- Revenue by Store and City
- Inventory Status
- Sales Trends Over Time

The project demonstrates practical SQL skills used in real data analyst roles.

---

## Database Used
PostgreSQL

---

## Database Schema

The database consists of the following tables:

1. toys_products
2. toys_stores
3. toys_inventory
4. toys_sales
5. calendar

---

## Key Business Questions Answered

### Data Exploration
- List all product categories
- Find stores in specific cities
- Count products per category

### Sales Analysis
- Total units sold
- Revenue per sale
- Revenue per store

### Business Metrics
- Top selling products
- Top revenue generating cities
- Stores with highest revenue

### Time Analysis
- Daily revenue
- Monthly revenue trends
- Best sales day

### Inventory Insights
- Products with low stock
- Products never sold

### Advanced SQL
- Window functions
- Running revenue totals
- Ranking products by revenue
